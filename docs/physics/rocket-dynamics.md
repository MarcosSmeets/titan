# Rocket Dynamics

## Overview

This document describes the physical models for rocket propulsion, mass variation, and multi-stage vehicles as implemented in the Titan Physics Engine.

## Governing Equations

Rocket motion is governed by Newton's Second Law applied to a variable-mass system:

```
F_total = m * a
a = (F_thrust + F_gravity + F_drag) / m
```

Position and velocity evolve as:

```
dr/dt = v
dv/dt = a(r, v, m, t)
```

## Thrust Model

Thrust is generated by expelling propellant mass at high velocity:

```
F_thrust = burn_rate * v_exhaust
```

Where:
- burn_rate (kg/s) - mass flow rate of propellant
- v_exhaust (m/s) - effective exhaust velocity

### Specific Impulse (Isp)

Isp measures engine efficiency in seconds:

```
Isp = v_exhaust / g0
```

Where g0 = 9.80665 m/s^2 (standard gravity).

| Engine Type | Isp (sea level) | Isp (vacuum) |
|------------|----------------|-------------|
| Solid booster | 240-270 s | 260-290 s |
| Kerolox (Merlin) | 282 s | 311 s |
| Hydrolox (RS-25) | 366 s | 452 s |
| Ion thruster | - | 3,000-12,000 s |

### Altitude-Dependent ISP

Titan supports ISP variation with altitude through the `ThrustForce` class:

```
Isp(h) = Isp_sea * (1 - rho/rho0) + Isp_vac * (rho/rho0)
```

This models nozzle expansion efficiency changes with ambient pressure.

## Mass Variation

Unlike most vehicles, rockets continuously lose mass during powered flight:

```
dm/dt = -burn_rate * throttle
m(t) = m_dry + m_fuel(t)
```

### Impact on Acceleration

As fuel is consumed:
- Total mass decreases
- Acceleration increases (for constant thrust)
- Thrust-to-weight ratio improves dynamically

This creates a nonlinear acceleration profile.

### Tsiolkovsky Rocket Equation

The theoretical delta-v from mass variation:

```
delta_v = v_exhaust * ln(m_initial / m_final)
```

Where:
- m_initial = dry mass + fuel mass
- m_final = dry mass (after burnout)

This is the fundamental equation of rocket propulsion.

## Thrust-to-Weight Ratio (TWR)

```
TWR = F_thrust / (m * g)
```

| TWR | Meaning |
|-----|---------|
| < 1.0 | Cannot lift off |
| 1.0 - 1.3 | Marginal (high gravity losses) |
| 1.3 - 2.0 | Optimal for 1st stage |
| 0.5 - 1.5 | Good for upper stages (vacuum) |
| > 4.0 | Excessive (high drag losses) |

## Multi-Stage Vehicles

Staging improves mass efficiency by discarding empty structural mass:

### Stage Separation Logic

```
if current_stage.fuel <= 0:
    discard current_stage (remove dry mass)
    activate next_stage
```

### Why Staging Works

Each stage carries only its own structural mass. After burnout, the empty stage (dry mass) is jettisoned, so subsequent stages don't waste energy accelerating dead weight.

For a two-stage rocket:

```
delta_v_total = delta_v_stage1 + delta_v_stage2
             = ve1 * ln(m01/mf1) + ve2 * ln(m02/mf2)
```

Where m02 is the mass of the rocket after stage 1 separation.

### Typical Stage Budget

| Stage | Delta-V | Purpose |
|-------|---------|---------|
| Stage 1 | 3,000 - 4,000 m/s | Liftoff through atmosphere |
| Stage 2 | 4,000 - 5,000 m/s | Orbit insertion |
| Stage 3 (if any) | 1,000 - 3,000 m/s | Circularization or TLI |

### Mass Fraction

The fuel fraction determines a stage's delta-v potential:

```
mass_fraction = m_fuel / (m_dry + m_fuel)
```

| Mass Fraction | Quality |
|-------------|---------|
| < 60% | Poor (heavy structure) |
| 60-80% | Acceptable |
| 80-90% | Good (modern rockets) |
| > 90% | Excellent (e.g. Starship) |

## Flight Phases

### 1. Powered Ascent

- Fuel > 0, thrust active
- Mass decreasing, acceleration increasing
- Gravity turn in progress (vertical to horizontal)

### 2. Ballistic Coast (if applicable)

- Between stages or after burnout
- No thrust, gravity and drag only
- Vehicle follows Keplerian trajectory

### 3. Circularization Burn

- At or near apoapsis
- Horizontal thrust to raise periapsis
- Achieves stable circular orbit

### 4. Orbit Insertion

- Periapsis > 180 km, eccentricity < 0.02
- Engine shutdown
- Mission complete
